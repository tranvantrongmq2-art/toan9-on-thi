🔥 Hướng dẫn cài đặt Firebase

⚠️ Nếu chưa cài Firebase, câu hỏi sẽ được lưu trên máy tính (localStorage). Khi đóng trình duyệt dữ liệu vẫn còn nhưng chỉ dùng được trên máy đó.
1

Tạo tài khoản Firebase

Truy cập console.firebase.google.com → Đăng nhập bằng Google → Tạo project mới

2

Tạo Firestore Database

Trong Firebase Console → Firestore Database → Create database → Chọn "Start in test mode" → Chọn vị trí (asia-southeast1)

3

Lấy cấu hình Firebase

Project Settings (⚙️) → Your apps → Web app → Add app → Sao chép config object

4

Điền config vào app.js

Mở file app.js → Tìm biến FIREBASE_CONFIG → Điền các giá trị từ bước 3

5

Bỏ comment Firebase SDK

Trong cả 3 file HTML, tìm phần comment <!-- Firebase SDK --> → Xóa dòng <!-- và -->

⚡ Nhập Firebase Config nhanh (dán từ Firebase Console)

* Config sẽ được lưu vào localStorage và áp dụng ngay lập tức. Để dùng vĩnh viễn, hãy chỉnh trực tiếp trong file app.js.
